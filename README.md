# Vanguard Spaces — Business Card Design

A modern and professional business card design built using **HTML**, **Tailwind CSS**, and **Google Fonts**. This project showcases a clean architectural studio brand identity with a two-sided business card layout.

## Preview

### Side A — Identity
- Company branding
- Minimalist design
- Premium typography
- Architectural studio identity

### Side B — Information
- Contact information
- Professional designation
- Studio location
- Website details
- Structured 12-column grid layout

## Features

- Responsive screen preview
- Modern minimalist aesthetic
- Tailwind CSS styling
- Google Fonts (Inter)
- Print-inspired business card proportions
- Front and back card designs
- Professional architecture-focused branding

## Technologies Used

- HTML5
- Tailwind CSS
- Google Fonts (Inter)
